# ⚙️ Installation

## <a name="serveur"></a> Serveur

- Construction du conteneur :
```shell
docker build -t juegos .
```
- Lancer le service :
```shell
docker run -d -p 8000:8000 juegos
```

La console devrait prévenir que le serveur est démarré.
